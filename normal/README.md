| 命令          | 用途          |
| ------------- | ------------- |
| .             | 重复上一次修改 |
| >             | 向右缩进      | 
| <             | 向左缩进      |
| ;             | 跟F/f 配合使用 （查找下一个）|
| ,             | 跟F/f 配合使用 （查找上一个）|
| dd            | 删除一行      |
| f{char}       | 在当前行查找字符 (向前)|
| F{char}       | 在当前行查找字符 (向后)|
| u             | 撤销上次修改  |
| x             | 删除一个字符  |

### 技巧
常用的组合技巧
#### [</>] 技巧
```vim
3>> #缩进三行
>G #从当前行到最后一行缩进
>gg #从当前行到最后一行缩进
>ap #缩进一段
```
